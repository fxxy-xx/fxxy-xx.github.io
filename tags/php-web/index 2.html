    序列化与反序列化

    • 序列化：在PHP中，序列化用于存储或传递 PHP 的值（比如数组或者类的值）的过程中，同时不丢失其类型和结构。就是将数组和类的值转换成字符串来传递。
    • 反序列化：将已经序列化的对象/数组的字符串还原成对象/数组格式。

    为什么要序列化？有利于存储或传递 PHP 的值，同时不丢失其类型和结构。
